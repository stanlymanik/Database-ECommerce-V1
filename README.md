## 📝 Deskripsi Proyek
Proyek ini merupakan perancangan basis data relasional yang komprehensif untuk platform E-Commerce. [cite_start]Sistem ini dirancang untuk menangani seluruh siklus operasional bisnis online, mulai dari manajemen pengguna, katalog produk, proses transaksi, hingga pelacakan pengiriman. [cite: 1, 4]

### Fitur Database:
* [cite_start]**13 Tabel Utama**: Mencakup Users, Produk, Pesanan, Pembayaran, Kurir, Pengiriman, dan lainnya. [cite: 4]
* [cite_start]**12 Views**: Laporan otomatis untuk stok rendah, riwayat belanja customer, dan produk terlaris. [cite: 4]
* [cite_start]**10 Stored Procedures**: Otomasi manajemen data produk dan pesanan untuk efisiensi sistem. [cite: 4]
* [cite_start]**Triggers & Logic**: Sistem cerdas yang otomatis mengurangi stok saat ada pesanan dan mengembalikan stok jika pesanan dibatalkan. [cite: 25]
